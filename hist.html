Nossa História

Saiba mais sobre o PetSave

Nós, alunos da turma de Técnico em Desenvolvimento de Sistemas da Unidade Senac Curvelo/MG, apresentamos como surgiu o projeto “PetSave”.

Com o desenvolver do nosso curso, realizamos um debate e uma votação com finalidade de fazer um projeto integrador voltado para a área social; projeto esse que vem com o objetivo de fazer o aluno colocar em prática tudo que aprendeu ao decorrer do curso.

Diante das ideias apresentadas pelos alunos, foi escolhido o projeto “PetSave” entre tantos outros projetos maravilhosos.

O projeto PetSave é um site totalmente pensado e criado pelos alunos com o objetivo de ajudar os animais vinculados com o Projeto Conexão Amor e Patas, a ONG Nupac (Núcleo de Proteção Animal de Curvelo).

Realizamos estudo de campo, pesquisas, entrevistas com os beneficiados dos projetos, e adquirimos os dados necessários para a divulgação no site “PetSave”.

Todos os alunos participaram do surgimento do projeto “PetSave”, alguns se foram mais deixaram sua colaboração. Todos os alunos se empenharam ao máximo para entregar o melhor trabalho para vocês.
